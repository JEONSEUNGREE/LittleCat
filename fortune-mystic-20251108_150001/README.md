# Fortune Mystic - 신비로운 운세 앱

타로 카드, 수정구슬, 오늘의 운세를 제공하는 신비로운 운세 웹 애플리케이션입니다.

## 주요 기능

- 🔮 **타로 카드 운세**: 12가지 타로 카드 중 하나를 선택하여 운세 확인
- 🌟 **수정구슬 점술**: 신비로운 수정구슬로 미래 예언
- ☀️ **오늘의 운세**: 하루 한 번 확인 가능한 일일 운세

## 기술 스택

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## 설치 및 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 특징

- 📱 모바일 퍼스트 반응형 디자인
- ✨ 애니메이션 효과 (카드 뒤집기, 빛나는 효과, 떠다니는 효과)
- 🎨 신비로운 보라색 + 금색 테마
- 💾 로컬 스토리지를 활용한 일일 운세 관리
- 🎭 운세 히스토리 기능
