# Virtual Pet Rock - 버추얼 펫 록

당신만의 버추얼 펫 록과 함께하는 명상 시간

## 특징

- 🗿 미니멀한 디자인의 가상 펫 록
- 🧘 명상과 휴식을 통한 상호작용
- 🎨 아름다운 그라데이션과 애니메이션
- 📱 모바일 최적화 반응형 디자인
- ✨ 감정 상태에 따른 비주얼 변화

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (아이콘)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 사용법

1. **Touch**: 펫 록을 터치하여 에너지 증가
2. **Meditate**: 명상하여 zen 상태 진입
3. **Rest**: 휴식하여 에너지 회복

## 진화 단계

- Pebble (조약돌)
- Stone (돌)
- Boulder (바위)
- Mountain (산)
