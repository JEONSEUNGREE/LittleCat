# Space Dodge Master

우주선을 조종하여 소행성을 피하는 액션 게임입니다.

## 기능
- 터치/마우스로 우주선 조종
- 점진적 난이도 증가
- 최고 점수 기록
- 반응형 디자인 (모바일 최적화)

## 기술 스택
- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태관리)
- Lucide React (아이콘)

## 실행 방법
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
```
