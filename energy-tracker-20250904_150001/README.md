# Energy Tracker

개인 에너지 레벨을 추적하고 최적의 작업 시간을 찾아주는 생산성 앱

## 기능

- 시간대별 에너지 레벨 기록
- 활동 및 기분 추적
- 24시간 에너지 패턴 분석
- 최적 작업 시간 추천
- 히스토리 관리

## 기술 스택

- React 18 + TypeScript
- Vite
- Tailwind CSS  
- Zustand (상태 관리)
- Lucide React (아이콘)

## 설치 및 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
