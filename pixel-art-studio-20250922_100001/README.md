# Pixel Art Studio

픽셀 아트를 그리는 창작 게임 앱

## 기능
- 8x8, 16x16, 24x24, 32x32 그리드 지원
- 펜, 지우개, 채우기, 스포이드 도구
- 실행 취소/다시 실행
- PNG 이미지로 내보내기
- 24가지 프리셋 색상 + 사용자 정의 색상
- 모바일 터치 지원

## 기술 스택
- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## 실행 방법
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
```
