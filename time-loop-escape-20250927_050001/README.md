# Time Loop Escape - 타임 루프 탈출

## 개요
시간 루프 탈출 게임 - 5초씩 반복되는 시간 속에서 올바른 행동 조합을 찾아 탈출하는 전략 퍼즐 게임

## 기술 스택
- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (State Management)
- Lucide React Icons

## 게임 특징
- 5초 타임 루프 메커니즘
- 점진적 퍼즐 해결
- 액션 시퀀스 전략
- 모바일 반응형 디자인

## 설치 및 실행
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
```
