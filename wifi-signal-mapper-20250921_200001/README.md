# WiFi Signal Mapper

실내 공간의 WiFi 신호 강도를 시각적으로 매핑하는 유틸리티 앱

## Features

- 📡 실시간 WiFi 신호 강도 측정
- 🗺️ 인터랙티브 히트맵 시각화
- 📊 네트워크 정보 및 통계
- 💾 측정 데이터 Export 기능
- 📱 모바일 최적화 반응형 디자인

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (State Management)
- Lucide React (Icons)

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
