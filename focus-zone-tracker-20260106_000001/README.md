# Focus Zone Tracker (집중 구역 추적기)

개인의 집중력 리듬을 추적하고 분석하여 최적의 작업 시간대를 제안하는 생산성 앱

## Features

- **실시간 집중 세션 추적**: 시작/종료로 집중 시간 기록
- **집중도 평가**: 세션 종료 시 집중 수준 자가 평가
- **시간대별 히트맵**: 24시간 집중 패턴 시각화
- **맞춤형 추천**: 개인의 집중 패턴 기반 최적 시간대 제안
- **데이터 영속성**: 로컬 스토리지에 세션 데이터 저장

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
