# 소비 심리 분석기 (Spend Psychology)

감정 기반 지출 추적 및 심리 분석 앱

## 주요 기능

- 💰 **지출 기록**: 금액, 카테고리, 감정 상태를 함께 기록
- 😊 **감정 분석**: 감정별 지출 패턴을 시각화
- 📊 **통계 분석**: 실시간 지출 통계 및 인사이트 제공
- 🎯 **심리 분석**: AI 기반 소비 심리 분석 및 피드백

## 기술 스택

- **Frontend**: Vite + React 18 + TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Icons**: Lucide React
- **Design**: Mobile-first responsive design

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 특징

1. **감정 기반 분석**: 5가지 감정 상태(행복, 평온, 슬픔, 분노, 스트레스)로 지출 추적
2. **실시간 시각화**: 감정별 지출을 바 차트로 시각화
3. **심리 인사이트**: 소비 패턴에 대한 심리학적 피드백 제공
4. **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 디바이스 지원
