# Mood Wave DJ

감정을 음악 파동으로 변환하는 인터랙티브 DJ 앱

## Features

- 5가지 감정 모드 선택 (Happy, Calm, Energetic, Sad, Neutral)
- 실시간 음악 파동 시각화
- DJ 컨트롤 (볼륨, 강도, 템포, 장르)
- 모바일 반응형 디자인

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
