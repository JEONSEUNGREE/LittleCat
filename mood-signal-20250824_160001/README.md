# Mood Signal - 실시간 감정 공유 소셜 앱

친구들과 현재 기분을 실시간으로 공유하고 서로 응원하는 소셜 플랫폼

## 주요 기능

- 🎨 8가지 감정 상태 선택 및 공유
- 👥 친구들의 실시간 감정 상태 확인
- 💬 감정 메시지 작성 기능
- ❤️ 친구 감정에 반응하기
- 📊 개인 감정 패턴 통계
- 📱 모바일 최적화 반응형 디자인

## 기술 스택

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
