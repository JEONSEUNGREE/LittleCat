# Focus Flow Timer

집중력을 높이는 시각적 포모도로 타이머

## 기능

- 🎯 포모도로 타이머 (25분 집중 / 5분 휴식)
- 🌊 집중도에 따라 변화하는 동적 배경 애니메이션
- 📊 세션 완료 및 플로우 강도 통계
- 🔔 타이머 완료 알림
- 📱 모바일 반응형 디자인

## 기술 스택

- React 18 + TypeScript
- Vite
- Zustand (상태 관리)
- Tailwind CSS
- Lucide React (아이콘)

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
