# Sound Wave Runner - 음파 리듬 러너

음파의 리듬을 타고 달리는 혁신적인 리듬 기반 러닝 게임

## 특징

- 🎵 음파 시각화와 리듬 게임의 융합
- 🌊 동적 음파 생성 시스템
- 🎮 직관적인 터치/클릭 조작
- 📱 모바일 최적화 반응형 디자인
- 🏆 점수 및 레벨 시스템
- ⚡ 부드러운 애니메이션과 시각 효과

## 기술 스택

- Vite + React 18
- TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## 실행 방법

```bash
npm install
npm run dev
```

## 게임 방법

1. 화면을 터치하거나 스페이스바를 눌러 점프
2. 음파의 리듬에 맞춰 장애물을 피하세요
3. 음파가 높을수록 더 높이 점프할 수 있습니다
4. 점수를 획득하고 레벨을 올려보세요!
