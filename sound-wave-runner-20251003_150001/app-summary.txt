====================================
앱 특징 및 차별점
====================================

기존 앱과의 차별점:
- 음악/리듬 기반 게임 (기존 앱들은 주로 물리/퍼즐 중심)
- 실시간 음파 시각화 엔진
- 동적 난이도 조절 시스템
- 리듬과 동기화된 장애물 생성

구현된 핵심 기능:
1. 실시간 음파 생성 및 시각화
2. 리듬 기반 점프 메커니즘
3. 동적 장애물 시스템
4. 레벨 진행 시스템
5. 점수 및 최고기록 저장
6. 반응형 모바일 UI

====================================
기술 구현 내역
====================================

구조:
- Vite + React 18 + TypeScript
- Tailwind CSS (반응형 디자인)
- Zustand (상태 관리)
- Canvas API (게임 렌더링)

컴포넌트:
- MainMenu.tsx: 메인 메뉴 화면
- GameCanvas.tsx: 게임 플레이 영역
- GameController.tsx: 게임 제어 로직
- GameOver.tsx: 게임 종료 화면
- gameStore.ts: 전역 상태 관리

====================================
실행 명령
====================================

npm install
npm run dev
