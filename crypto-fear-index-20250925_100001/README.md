# Crypto Fear & Greed Index

실시간 암호화폐 시장 심리 분석 앱

## 기능

- 실시간 공포/탐욕 지수 표시 (0-100)
- 시각적 미터 게이지
- 과거 데이터 차트
- 시장 인사이트 및 투자 전략 제안
- 반응형 모바일 우선 디자인

## 기술 스택

- Vite + React 18 + TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- Lucide React (아이콘)

## 설치 및 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
